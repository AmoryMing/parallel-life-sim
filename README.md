# 平行人生模拟器

> "不是准确，是仿真；不是预测，是说服。"

基于个人数据的人生仿真引擎。输入你的真实经历和性格，AI 为你构建一个有记忆、有性格的"数字分身"。改变一个条件——换个城市、换个专业、换个时代——看另一个版本的你会怎么活。你可以和30岁的自己对话，看50岁的自己写的墓志铭，甚至让两个平行宇宙的自己吵一架。不是算命，不美化人生。它的价值不在准确，在于让你看见那些你永远走不了的路。

## 核心功能

- **造** — 对话式采集个人数据，AI 生成人格档案和变量面板
- **养** — AI 自动推演人生，用户控制时间流速，观察不同版本的自己
- **悟** — 通过观察、对话、凭吊，反过来理解真正的自己

## 本地部署

```bash
# 克隆仓库
git clone https://github.com/AmoryMing/parallel-life-sim.git
cd parallel-life-sim

# 创建虚拟环境
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install -r requirements.txt

# 配置 API Key（支持 OpenAI 兼容协议的任何大模型）
# 在 app.py 中修改 base_url 和 api_key

# 启动
python3 app.py
```

打开浏览器访问 http://localhost:5678

## 使用流程

1. **认识你** — 在对话框里介绍自己，AI 会追问细节
2. **你的原本** — 查看 AI 生成的人格档案和变量面板，可以编辑修改
3. **平行人生** — 选择模拟预设（如实推演/悲观线/聚焦事业），开始模拟
4. 模拟完成后可以从任意节点**创建分支**，改变一个条件重新跑

## 技术栈

- **后端**: Python + Flask
- **前端**: 原生 HTML/CSS/JS（单文件）
- **AI**: 支持 OpenAI 兼容 API（默认千问 qwen3.5-plus）
- **设计风格**: 浅色主题，参考 Notion + Claude
