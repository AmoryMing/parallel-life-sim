# Changelog

## v0.1.2 — 2026-03-14（前端重设计 + 社会变量框架）

**触发：** 用户要求浅色主题、参考 Notion/Claude 风格；社会变量不够真实

### 改动

1. **前端浅色主题**（`templates/index.html`）
   - 暗色 → 浅色：米白底 #fafaf8、温暖琥珀强调色 #b5651d
   - 设计参考 Notion（干净排版、留白）+ Claude（温暖色调、圆角、柔和阴影）
   - 用户消息气泡：米色背景 + 琥珀边框；AI 消息：无背景直排
   - 弹窗遮罩：backdrop-filter blur + 半透明灰

2. **社会变量框架**（新文件 `社会变量框架.md`）
   - 5层32个变量：个人(7) → 人际(6) → 社会(6) → 国家(6) → 国际(5)
   - 所有变量基于 2026 年中国真实数据（两会报告、研报、行业数据）
   - 含 5 条变量传导链和模拟引擎设计建议

## v0.1.1 — 2026-03-14（第一轮用户反馈修复）

**触发：** 用户试用 MVP 后的 5 条反馈

### 改动

1. **AI 主动开场白**（`templates/index.html`）
   - 问题：进入页面是空白的，用户不知道该干什么
   - 修复：页面加载时 AI 自动发送欢迎语引导用户开口

2. **人格档案可编辑**（`templates/index.html` + `app.py`）
   - 问题：生成的人格档案不能修改，用户觉得不准也没法改
   - 修复：加了"编辑"按钮，可以直接修改 Markdown 原文，保存后存到 `data/persona.md`

3. **模拟预设按钮替代"附加指令"**（`templates/index.html`）
   - 问题：用户不知道"附加指令"是什么意思，不知道怎么用
   - 修复：加了 4 个预设按钮（如实推演/悲观线/聚焦事业/聚焦感情），同时保留自由输入

4. **面试流程优化：优先追问基础信息**（`prompts.py`）
   - 问题：AI 没有主动问性别等基础信息，导致人格档案缺失关键数据
   - 修复：提示词增加优先级，基础信息（性别、年龄等）排第一，第二轮必须追问补齐

5. **社会变量真实性**（待做）
   - 问题：社会变量太泛、太个人化，缺少基于真实社会现象的分析
   - 状态：需要进一步调研，暂未修改
